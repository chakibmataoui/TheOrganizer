<h1 style="color:red">THE ORGANIZER</h1>

<p>The organizer is a simple utility program that helps organizing file into different folders.
It's easy to use , you just need with right-clicking on a folder or in the background of the file explorer when you're inside a desktop (Or even in the background of the desktop) and choose which way you want things to be organized 
For the desktop don't worry, the icons and hidden files will stay safe they don't move </p>
